# Lotto Number Recommendation Site

## Overview

This is a simple web application that generates and displays recommended lottery numbers.

## Features

*   Generates 6 unique random numbers between 1 and 45.
*   Displays the numbers in a visually appealing format.
*   Allows the user to generate a new set of numbers with a button click.
